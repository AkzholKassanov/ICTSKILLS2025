Компетенция: Робототехника

“ICT Skills 2025”

Компетенция “Робототехника” определяется как умение участников самостоятельно собирать схему устройства, написать код для микроконтроллера и протестировать предложенными данными. Участник должен будет собрать устройство по предложенной схеме на базе робототехнической платформы Arduino Uno и написать программу для управления в среде разработки Arduino IDE.

Требования

Необходимые знания:

  • основы схемотехники и электроники для “чтения” принципиальных схем;
  • создание и сборка устройства на основе схемы;
  • навыки программирования на языке C++ для написания программы.

Описание задания

В ходе выполнения конкурсного задания участникам предстоит собрать устройство на основе тех элементов набора и ресурсов, которые будут предоставлены организаторами.

Компетенция: Веб-разработка: frontend / backend

"ICT Skills 2025"

Компетенция «Веб-разработка: Frontend / Backend» — это совокупность навыков, необходимых для создания полнофункционального веб-сайта, включающего пользовательский интерфейс (frontend) и серверную логику (backend). Разработка осуществляется с использованием HTML, CSS, JavaScript, PHP и MySQL. Конкурс проводится для учителей общеобразовательных школ, желающих освоить современные инструменты веб-программирования, применить их в педагогической практике, получить экспертную оценку своей работы и заряд вдохновения от коллег.

Требования

Необходимые знания и навыки:

  • уверенное владение HTML для создания структуры страниц;
  • знание CSS для стилизации и адаптивной верстки;
  • основы JavaScript для реализации интерактивности на сайте;
  • использование PHP для обработки запросов и взаимодействия с сервером;
  • умение подключаться и работать с базой данных MySQL (создание таблиц, выполнение запросов);
  • базовые знания клиент-серверной архитектуры;
  • создание форм, работа с отправкой и обработкой данных;
  • понимание принципов хранения и извлечения информации из БД.

Описание задания

В ходе выполнения конкурсного задания участникам предстоит разработать веб-сайт с интеграцией клиентской и серверной части. Проект должен демонстрировать целостный подход к созданию веб-приложения: от визуального оформления до работы с данными и их хранением. Работа должна отражать умение использовать основные технологии веб-разработки для решения прикладной задачи.

Компетенция: Проектирование и моделирование сетей

“ICT Skills 2025”

Компетенция “Проектирование и моделирование сетей” определяется как умение участников самостоятельно строить локальную сеть, используя востребованные топологии сетей в модели OSI, а также их настройку с использованием статических/динамических IP-адресов с помощью различных сетевых устройств (hub, switch и router).

Требования

Необходимые знания:

  • разновидности сетевых устройств и их назначения;
  • стеки протоколов TCP/IP;
  • команды, используемые для диагностики сети.

Необходимые умения:

  • знать основы использования Cisco Packet Tracer;
  • строить локальные сети на основе проводных/беспроводных технологий.

Пример задания

Используя Cisco Packet Tracer, построить локальную сеть. К локальной сети подключить сетевой принтер с возможностью печати по сети через любое подключенное к сети устройство (ПК или ноутбук).

Проверка умений использовать утилиты:

  • проверки целостности и качества соединений в сетях на основе TCP/IP;
  • трассировку маршрута до заданного узла в локальной сети;
  • настройку беспроводной сети с использованием сетевого оборудования.

Компетенция: Олимпиадное программирование

“ICT Skills 2025”

Компетенция “Олимпиадное программирование” позволяет проверить навыки программирования через решение практических, нестандартных заданий. Программист решает задания посредством решения задач с использованием языка программирования по выбору или по выбору организатора.

Требования

Необходимые знания и умения:

  • уметь читать псевдокод;
  • разрабатывать свое решение с использованием разных операторов;
  • тестировать свою программу, используя входные и выходные данные.

Описание задания

В ходе выполнения конкурсного задания участникам предстоит решить задачи разных уровней, написав свои алгоритмы для решения данных задач. Участник может использовать любые дозволенные программные конструкции и операторы для достижения результата. Проверка написанных программ будет производиться по нескольким критериям, включая быстроту выполнения и успешное прохождение нескольких уровней тестов с заданными входными параметрами.

Компетенция: Разработка мобильных приложений

Компетенция определяется как набор профессиональных навыков процесса разработки мобильного приложения, редактирования, использования дружественного интерфейса и обеспечения адаптивности. Для работы используются программно-вычислительные средства (персональные компьютеры, мобильные телефоны с ОС Android).

Конкурс проводится для учителей общеобразовательных школ, которые хотят:

  • сделать что-нибудь новое и интересное;
  • получить опыт и советы экспертов;
  • получить большое количество контактов и обратную связь от других участников;
  • получить «крутые» эмоции.

Требования

Необходимые знания:

  • знание блочного программирования;
  • создание приложений от кода и проекта до дизайна интерфейса;
  • владение базой объектно-ориентированного программирования;
  • настройка и тестирование приложения;
  • работа с базами данных;
  • умение писать, читать и редактировать код;
  • выстраивание архитектуры приложения.

Описание задания

В ходе выполнения конкурсного задания участникам предстоит спроектировать мобильное приложение с использованием различных видов алгоритма.
